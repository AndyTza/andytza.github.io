## The SDSS Stripe-82 Variable Source Catalog



The data presented here has been taken directly from [IveziÄ‡ et al. 2007](http://faculty.washington.edu/ivezic/sdss/catalogs/205821.web.pdf). Detailed data description can be found [here](http://faculty.washington.edu/ivezic/sdss/catalogs/S82variables.html).

This is part of an academic project, if you are using this dataset please ensure you use proper citation:
